# Vodafone Cards - Android App
## ⌯ Virus Net ⌯

تطبيق Android مبني من كود PHP للوصول إلى كروت فودافون.

## كيفية البناء

1. ارفع المشروع على GitHub
2. اذهب إلى **Actions** → ستجد workflow جاهز
3. اضغط **Run workflow**
4. بعد الانتهاء اضغط على **Artifacts** وحمل الـ APK

## الملفات
- `index.html` - صفحة تسجيل الدخول
- `cards.html` - عرض الكروت
- `dev_room.html` - الغرفة السرية
